The LoCloud Geocoding Microservice is a lightweight SaaS Cloud application that makes it easy to add coordinates to any existing tabular data.

The application is designed to be easy to deploy on the most common hosting platforms and is built exclusively using Open Source software components.

Open Standards

The internal communication in the application is based on the following Open Standards

  • HTML5
  • JSON, GeoJSON
  • KML
  • JavaScript
  • jQuery, jQeryUI
  • OpenLayers
  • Proj4JS

Furthermore, the application makes use of online spatial data services both as searchable databases and map backgrounds

  • Google Maps API v3
  • BING Maps
  • OpenStreetMap
  • GeoNames
  • WMS services (capable of connecting to any INSPIRE compliant web map server)

The solution is capable of running on Linux/Windows - Apache - MySQL and PHP (LAMP/WAMP) platforms meeting the below requirements:

  • PHP: version >= 5.3
  • MySQL: version >= 5
  • Apache: version >= 2
